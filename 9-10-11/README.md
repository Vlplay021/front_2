# Магазин с JWT аутентификацией и RBAC

Данный проект представляет собой веб-приложение интернет-магазина с системой аутентификации на основе JSON Web Tokens (JWT), поддержкой refresh-токенов и разграничением прав доступа (RBAC). Приложение состоит из бэкенда на Node.js (Express) и фронтенда на React.

## Технологии

- **Backend**: Node.js, Express, bcrypt (хэширование паролей), jsonwebtoken, nanoid, cors
- **Frontend**: React, axios, Font Awesome
- **Аутентификация**: JWT access + refresh токены
- **Управление ролями**: три роли – `admin`, `seller`, `user`
- **Хранение данных**: in-memory (для демонстрации)

## Основные возможности

- Регистрация новых пользователей (роль по умолчанию – `user`)
- Вход с получением access и refresh токенов
- Обновление access токена с помощью refresh токена
- Просмотр списка товаров для всех авторизованных пользователей
- Создание и редактирование товаров для пользователей с ролями `seller` и `admin`
- Удаление товаров только для `admin`
- Управление пользователями (просмотр, редактирование, блокировка) – только для `admin`
- Поддержка изображений товаров (URL картинки)

## Запуск проекта

### Требования
- Node.js (версия 14+)
- npm или yarn
