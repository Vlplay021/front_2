# PWA “Заметки” (WebSocket + Push)

## Основной функционал
- Офлайн-доступ через Service Worker (кэширование статики и страниц)
- Установка на устройство (Web App Manifest)
- Синхронизация заметок в реальном времени (Socket.IO)
- Push-уведомления с напоминаниями и возможностью отложить на 5 минут

## Структура проекта
| Файл | Роль |
|------|------|
| `index.html` | Главная страница, навигация, подключение стилей/скриптов |
| `app.js` | Клиентская логика: навигация, localStorage, WebSocket, Push API |
| `server.js` | Сервер (Express + Socket.IO + web-push) |
| `sw.js` | Service Worker: кэширование, обработка push и кликов |
| `manifest.json` | PWA-манифест (имя, иконки, тема) |
| `content/*.html` | Шаблоны страниц (Home, About) |

## Ключевые моменты
1. **Офлайн** – SW кэширует `/`, `index.html`, `app.js`, иконки и динамические страницы `/content/`.
2. **WebSocket** – при добавлении заметки клиент отправляет `newTask`, сервер рассылает `taskAdded` всем.
3. **Push + напоминания** – клиент подписывается через `PushManager`, сервер хранит таймеры. По истечении времени отправляет push. В уведомлении есть кнопка "Отложить" → запрос на `/snooze` → перенос на +5 минут.
4. **Хранилище** – заметки лежат в `localStorage`, синхронизация через WebSocket не перезаписывает чужие записи (проверка по `id`).

## Запуск
```bash
npm install express socket.io web-push body-parser cors
node server.js   # порт 3001
```
Открыть http://localhost:3001. Для push обязательно HTTPS (или localhost).
